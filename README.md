## ✨ Можливості

| Функція | Опис |
|---------|------|
| 🚀 **Завантаження** | 3-секундний анімований екран |
| 🔒 **Блокування** | Годинник з датою, свайп для розблокування |
| 📞 **Телефон** | Робоча клавіатура, кнопка CALL з підтвердженням |
| 💬 **SMS** | Список чатів, кнопка нового повідомлення |
| 📷 **Камера** | Відошукач, кнопка зйомки (симуляція) |
| ⚙️ **Налаштування** | Wi-Fi, Bluetooth, сховище, "Про телефон" |
| 🎵 **Музика** | Плеєр з кнопками керування |
| 📨 **Email** | Вхідні повідомлення (демо) |
| 🔋 **Статус бар** | Час, батарея, назва пристрою |
